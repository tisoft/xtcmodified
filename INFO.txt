  Infos:
  ------------------------------------------------------------------------
  xt:Commerce ist eine Weiterentwicklung der bekannten Shop-software osCommerce.
  Der xt:Commerce Shop legt ein besonderes Augenmerk auf Sicherheitsrelevante 
  Aspekte,Usability und ist durch einen neuen professionellen Aufbau gekennzeichnet.
  
  xt:Commerce ist auf jedem System lauffaehig, welches eine PHP Umgebung
  (ab PHP 4.1.3) und mySQL zur Verfuegung stellt, wie zum Beispiel Linux, 
  Solaris, BSD, und Microsoft Windows.
 
  Weiters sollte am Server installiert sein:

  GDlib2+ 
  PDFlib
